# ⚡ FX Power Calculator

A simple and powerful web app designed for **event, club, and production teams** to calculate electrical load quickly and safely.

---

## 🚀 Features

- ➕ Add unlimited equipment
- ⚡ Input in **Watts or Amps**
- 🔄 Auto-convert Watt → Amp
- 📍 Assign equipment to zones:
  - Stage
  - Entrance
  - Booth
  - Truss
  - Others (custom input supported)
- 🧮 Real-time total load calculation
- 🔌 3-phase power estimation
- 🛡️ Safety mode (2x power calculation)

---

## 🎯 Purpose

This tool is built for:

- Event organizers
- DJs & production crews
- Lighting & special effects teams
- Technical managers

👉 Helps prevent:
- Power overload
- Equipment failure
- Safety risks at events

---

## 🧠 How It Works

1. Add equipment name
2. Enter Watt or Amp
3. Select zone
4. App calculates:
   - Total Amp
   - Required 3-phase power
   - Safety-adjusted load
